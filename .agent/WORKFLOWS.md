# Intro

List of workflows all AI agents must follow.

## General Guide

Before AI agent proceed with a task, it must formulate the correct very tiny & very detailed steps to follow, by analyzing all listed workflows. This means generate an internal TO-DO list which contains, in correct order and combination, the steps of listed workflows.
