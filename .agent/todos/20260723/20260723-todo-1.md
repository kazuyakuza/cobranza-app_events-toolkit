# TODO: Rename `GatewayConsumerOptions` to `ModuleConsumerOptions`

## Context

The interface `GatewayConsumerOptions` was introduced in v0.14.0 to hold top-level consumer defaults passed through `EventsToolkitModule.forRoot()`. The word **gateway** was inherited from the original TODO which referenced `ms-db-gateway` as the downstream service. This creates an unwanted coupling between the library's public API and a specific microservice name. The name should be decoupled to avoid confusion.
