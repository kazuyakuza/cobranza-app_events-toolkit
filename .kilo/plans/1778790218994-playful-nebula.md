### 1.2 Add YAML Frontmatter to Command Files
Frontmatter gives commands metadata Kilo uses in the UI:

```yaml
---
description: Critical workflow for task execution
agent: plan
---
```

- `description`: Shown in command picker `/` menu
- `agent`: Which agent handles this command (`plan` for orchestration)
- `model`: Optional model override
- `subtask`: When `true`, runs as isolated sub-agent session
